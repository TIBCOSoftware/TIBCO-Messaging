1. Make sure you have Python 3 and pip installed.
2. pip install eftl-1.0.0.tar.gz
3. Set the environment variables TIBCO_TEST_URL and TIBCO_TEST_PASSWORD.
	   To use the 'twisted' library, set the environment variable TIBCO_TEST_TWISTED.
4. Run demo_receive.py
5. Run demo_send.py
       If you only want to send one message, you may type it with or without quotes as command line 
       arguments.
6. These demos run indefinitely, so issue a keyboard interrupt to stop.
